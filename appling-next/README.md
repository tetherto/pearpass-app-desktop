# PearPass Appling

Pear Appling installer for PearPass Desktop application.

## Setup

Setup is moving to pear, for now install bare-build.
See bare-build for more detailed instructions for universal mac builds, signing etc.

For quick demos do

```sh
bare-build --target=darwin-arm64 --icon lib/icons/darwin/icon.png app.cjs
```
